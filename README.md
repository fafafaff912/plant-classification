# Автоматическая классификация сортов растений по изображениям семян или листьев

##  Описание задачи
Разработка модели компьютерного зрения для мультиклассовой классификации сортов растений на основе фотографий семян или листьев. Задача решается методами глубокого обучения с использованием свёрточных нейронных сетей (CNN) или Vision Transformer (ViT).

##  Цель и мотивация
Автоматизация определения сорта растений позволяет ускорить работу ботаников, селекционеров и сельскохозяйственных специалистов. В проекте исследуются две современные архитектуры: ResNet50 (классическая CNN) и Vision Transformer (ViT) — с целью сравнения их эффективности на ограниченном наборе данных.

##  Использованные данные
- **Seed Image Dataset** (Kaggle): [ссылка](https://www.kaggle.com/datasets) — изображения семян 14 сортов.
- **Leafsnap Dataset**: [ссылка](http://leafsnap.com/dataset/) — изображения листьев 185 видов деревьев.  
В проекте использована выборка из 10 классов с балансировкой.

##  Архитектуры моделей
1. **ResNet50** (предобученная на ImageNet) с заменой последнего полносвязного слоя.  
   *Обоснование*: хорошо зарекомендовала себя для классификации изображений, устойчива к переобучению благодаря residual-блокам.
2. **Vision Transformer (ViT)** (предобученный) из библиотеки `timm`.  
   *Обоснование*: современная архитектура на основе attention, способная конкурировать с CNN.

##  Метрики качества
- Общая точность (accuracy)
- Точность по каждому классу (per-class accuracy)
- Матрица ошибок (confusion matrix)
- Полнота, точность, F1-мера (classification report)

##  Результаты
| Модель       | Test Accuracy | Per-class accuracy (min/mean/max) |
|--------------|---------------|-----------------------------------|
| ResNet50     | 0.934         | 0.85 / 0.93 / 1.00                |
| ViT          | 0.951         | 0.90 / 0.95 / 1.00                |

Графики и матрицы ошибок доступны в папке `reports/`.
